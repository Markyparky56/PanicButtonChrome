NoFap Panic Button – Privacy Policy

Last Updated: 2018/11/06

– What Information Do We Collect, Why, and How Is It Stored?
The NoFap Panic Button does not collect and store any of your personal data.

Elements of the NoFap Panic Button (which may be disabled) require access to your browsing data so that they may parse the web pages you are visiting, this processing is required for those elements of the extension to function. This data is not transmitted externally.

Of note: the NoFap Panic Button feature NSFWGuard maintains a local database of nsfw subreddits. If NSFWGuard encounters a subreddit during browsing which is not contained in this database it will add it to the database. This database is entirely local to your computer and is not communicated to NoFap LLC or any third-parties.

The NoFap Panic Button provides an options menu which allows the user to configure which additional features of the extension they want. These options are stored via Google Chrome’s storage sync API, which will synchronise options across devices. We only store configuration options via the sync API, including which features are enabled, NSFWGuard’s redirect link, the side the Popup Panic Button appears on and the Popup Panic Button’s site Blacklist.

– Do We Use Any Cookies?
No, the NoFap Panic Button Extension does not set or read any cookies during your browsing experience.

– Information Sharing
NoFap LLC values your privacy, as such we do not transmit any data regarding your use of the extension or your browsing habits.

– Dispute Resolution
If you have a complaint about the NoFap Panic Button extension you should contact NoFap LLC at contact@NoFap.com. Our mailing address is PO Box 81015, Pittsburgh PA 15217.

– Changes To This Policy
We may update this Privacy Policy to reflect changes in how we handle any data used by the extension, in light of changes to the functionality of the extension. If we make any changes you will be notified of those changes through the extension and offered the opportunity to disable any features which might be related to those changes or to uninstall the extension.
